PHP 7.1

With the excellent new PHP 7.0 out and about for over half a year already, PHP 7.1 is heading to a server near you, and it has some really cool stuff! PHP 7.1 is only a beta right now, but you can keep an eye on this page to find out when exactly PHP 7.1 is ready for use.

If you want something stable, stick to PHP 7.0 with help from PHP The Right Way: Getting Started, or you can get a bit more adventurous and install whichever version you like from source.
